### 2.2 Tipografía

#### Familia Tipográfica Principal

**MONTSERRAT** (Títulos y encabezados)
- **Pesos**: Bold (700), SemiBold (600), Medium (500)
- **Uso**: H1, H2, H3, títulos de sección, CTAs
- **Características**: Geométrica, moderna, técnica
- **Tamaños**:
  - H1: 48px / 3rem (Desktop) | 36px / 2.25rem (Mobile)
  - H2: 36px / 2.25rem (Desktop) | 28px / 1.75rem (Mobile)
  - H3: 28px / 1.75rem (Desktop) | 24px / 1.5rem (Mobile)
  - H4: 24px / 1.5rem (Desktop) | 20px / 1.25rem (Mobile)

**INTER** (Cuerpo y lectura)
- **Pesos**: Regular (400), Medium (500), SemiBold (600)
- **Uso**: Párrafos, descripciones, listas, textos largos
- **Características**: Legible, versátil, optimizada para pantallas
- **Tamaños**:
  - Body Large: 18px / 1.125rem
  - Body Regular: 16px / 1rem
  - Body Small: 14px / 0.875rem
  - Caption: 12px / 0.75rem

#### Jerarquía Tipográfica


## 3. SISTEMA DE DISEÑO

### 3.1 Espaciado y Grid

#### Sistema de Espaciado (8px base)
```
4px  = 0.25rem (micro)
8px  = 0.5rem  (xs)
16px = 1rem    (sm)
24px = 1.5rem  (md)
32px = 2rem    (lg)
48px = 3rem    (xl)
64px = 4rem    (2xl)
96px = 6rem    (3xl)
```

#### Grid System
- **Desktop**: 12 columnas | Gap: 24px | Max-width: 1280px
- **Tablet**: 8 columnas | Gap: 16px | Max-width: 768px
- **Mobile**: 4 columnas | Gap: 16px | Max-width: 375px

### 3.3 Animaciones y Transiciones

#### Principios de Animación
- **Duración**: 0.2s (rápidas), 0.3s (estándar), 0.5s (lentas)
- **Easing**: `cubic-bezier(0.4, 0, 0.2, 1)` (ease-in-out)
- **Propósito**: Feedback visual, jerarquía, fluidez

#### Animaciones Definidas
```css
/* Fade In */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* Slide In Up */
@keyframes slideInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Scale In */
@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}

/* Pulse Glow */
@keyframes pulseGlow {
  0%, 100% {
    box-shadow: 0 0 20px rgba(255,122,0,0.3);
  }
  50% {
    box-shadow: 0 0 40px rgba(255,122,0,0.6);
  }
}
```

## 6. TONO DE VOZ Y MESSAGING

### 6.1 Personalidad de Marca

**Atributos de Voz**:
- **Técnico pero Accesible**: Usamos terminología correcta, pero la explicamos
- **Confiado**: Somos expertos, lo sabemos, pero no somos arrogantes
- **Directo**: Sin rodeos, al grano, valor inmediato
- **Bilingüe Nativo**: No "traducimos", pensamos en ambos idiomas

### 6.2 Principios de Copywriting

#### Estructura de Mensajes
1. **Hook**: Captura atención con dato o pregunta
2. **Problema**: Identifica pain point del usuario
3. **Solución**: Presentamos nuestra propuesta de valor
4. **CTA**: Acción clara, específica, urgente

#### Ejemplos de Tono

**❌ Evitar**:
> "En WeldTech Solutions nos dedicamos a proveer servicios de información relacionados con los procesos de calificación de soldadores..."

**✓ Preferir**:
> "¿Necesitas cualificar soldadores según ASME IX? Descarga nuestro esquema completo y ahorra horas de revisión de normativas."

#### Vocabulario Técnico
- **Usar**: Cualificación, WPS, PQR, espesor cualificado, variables esenciales
- **Evitar**: Jerga innecesaria, anglicismos sin traducción, tecnicismos sin contexto

### 6.3 Mensajes Clave (Key Messages)

**Propuesta de Valor Principal**:
> "Transformamos normativas complejas en herramientas prácticas para la cualificación de soldadores con trazabilidad total."

**Mensajes Secundarios**:
1. "Cumplimiento garantizado con ASME IX, AWS D1.1 e ISO 9606"
2. "Información bilingüe: aprende y aplica en inglés y español"
3. "Reducción de hasta 40% en tiempo de revisión documental"
4. "Recursos diseñados por inspectores certificados CWI/CSWIP"

## 7. ESTRATEGIA DE ESCALABILIDAD

### 7.1 Fases de Crecimiento

#### FASE 1: Fundación (Actual)
**Alcance**: Infografía única, landing page, WhatsApp
**Visual**: Core identity establecida (colores, fuentes, estilo)
**Contenido**: Foco en ASME IX
**Aplicaciones**: Digital únicamente

#### FASE 2: Expansión (3-6 meses)
**Alcance**: Serie de infografías (AWS D1.1, ISO 9606, etc.)
**Visual**: Introducción de subsistemas por estándar
**Contenido**: Cursos cortos, webinars
**Aplicaciones**: Introducción de print materials

**Nuevos Elementos Visuales**:
- **Códigos de color por estándar**:
  - ASME IX: Orange (#FF7A00)
  - AWS D1.1: Blue (#2AA1FF)
  - ISO 9606: Green (#10B981)
  - API 1104: Purple (#8B5CF6)
- **Templates diferenciados**: Mantener core identity pero con variación

#### FASE 3: Consolidación (6-12 meses)
**Alcance**: Plataforma SaaS, membresías, comunidad
**Visual**: Sistema de diseño completo (Design System formal)
**Contenido**: Biblioteca completa de recursos
**Aplicaciones**: Multi-plataforma (web, mobile app)

**Nuevos Componentes**:
- Dashboard interfaces
- Data visualization styles
- Video content templates
- Podcast branding

#### FASE 4: Liderazgo (12+ meses)
**Alcance**: Referencia internacional, certificaciones propias
**Visual**: Evolución natural sin perder esencia
**Contenido**: Marketplace de recursos, API para terceros
**Aplicaciones**: White-label solutions

### 7.2 Consistencia en Crecimiento

#### Elementos Inmutables (Never Change)
- ✓ Paleta de colores core (Dark Steel, Weld Orange, Precision Blue)
- ✓ Tipografía principal (Montserrat + Inter)
- ✓ Estilo de iconografía (line icons, 2px)
- ✓ Tono de voz (técnico, directo, confiado)

#### Elementos Flexibles (Can Evolve)
- ~ Paleta complementaria (se puede expandir)
- ~ Fotografía y elementos visuales (pueden sofisticarse)
- ~ Animaciones (pueden hacerse más complejas)
- ~ Aplicaciones de marca (nuevos medios)

#### Gobernanza de Marca
**Responsable**: Creative Director / Brand Manager
**Revisión**: Trimestral para ajustes menores, anual para evoluciones mayores
**Aprobaciones**: Todo material nuevo debe seguir este manual o proponer actualización formal

**Inspiración de Estilo**:
- Diseño industrial: Siemens, ABB, Schneider Electric
- Plataformas técnicas: GitHub, Linear, Vercel

**No Emular**:
- Diseño corporativo tradicional (aburrido, genérico)
- Tech startup saturado (demasiado colorful, sin seriedad)
- Institucional académico (demasiado formal, poco dinámico)
