## Cómo Capturar Screenshots

### Desktop (1920x1080)

```bash
# Usar herramienta de captura del navegador
# Chrome DevTools → Cmd/Ctrl + Shift + P → "Capture screenshot"
```

### Mobile y Tablet

```bash
# Chrome DevTools → Toggle Device Toolbar (Cmd/Ctrl + Shift + M)
# Seleccionar dispositivo (iPhone 12, iPad, etc.)
# Capturar screenshot
```

### Optimizar Imágenes

```bash
# Usar TinyPNG o herramientas similares
# Formato: PNG
# Comprimir para web
```
